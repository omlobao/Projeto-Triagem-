**Projeto final do curso Desenvolvedor Python do Senai-DF Unidade Taguatinga**

O projeto tem como objetivo criar um sistema de triagem hospitalar funcional, com cadastro de pacientes, enfermeiros médicos e usuários além de CRUD's básicas.
